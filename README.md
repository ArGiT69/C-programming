## 🚀 Getting Started

### Prerequisites
- A C compiler (GCC, Clang, or MSVC)
- Text editor or IDE (VS Code recommended)

### Compiling and Running

Compile a program:
```bash
gcc first.c -o first
```

Run the compiled program:
```bash
./first        # Linux/Mac
first.exe      # Windows
```

Or compile and run in one step:
```bash
gcc first.c && ./first.exe  # Windows
gcc first.c && ./a.out      # Linux/Mac
```
